Pastikan setelah mengambil projek ini dari github, masukan perintah:

npm install

1. Selanjutnya, pastikan sudah masuk ke folder aoca-session-20-master (bagian front-end)
2. Lalu jalankan di terminal:

   npx expo start -c atau npx expo start

3. Lalu untuk bagian backend, tambahkan terminal baru di vs code dan masuk ke folder backend dengan cara:

   cd backend

4. setelah itu jalankan backend dengan cara:

   npm run dev
